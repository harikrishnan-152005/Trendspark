# рџ§  Backend Setup

Navigate to backend folder:

```

```

Create virtual environment:

```
python -m venv venv
```

Activate environment:

Windows

```

```

Install dependencies:

```
pip install -r requirements.txt
```

Run the API server:

```
uvicorn main:app --reload
```

Backend server runs at:

```
http://127.0.0.1:8000
```
